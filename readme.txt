== Description ==

This plug-in generates a report file from the post .
The report file is made from Excel file that is uploaded by you.
You can set customfield's value to Excel worksheet cell.

Supported formats are XLS, XLSx.

= FOSS Disclaimer =
One opensource libraries are included with this plugin (without changes)
> PHPExcel version 1.8.0, 2014-03-02 - http://phpexcel.codeplex.com


== Installation ==

1. Upload the contents of the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open Setting -> Excel Reports
4. Check posttype you want generate report from, and save.
3. You can find new custom posttype 'Excel Reports'.
   Entry new Excel report.

For basic usage, you can also have a look at the [plugin homepage](http://www.eyeta.jp/archives/1040).
